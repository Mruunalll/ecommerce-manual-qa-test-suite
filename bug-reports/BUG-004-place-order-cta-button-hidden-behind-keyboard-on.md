## BUG-004 — "Place Order" CTA button hidden behind keyboard on iOS Safari checkout

**Title:** "Place Order" CTA button hidden behind keyboard on iOS Safari checkout  
**Environment:** iPhone 14, iOS 17, Safari, WooCommerce Storefront theme  
**Severity:** Critical | **Priority:** High  

**Steps to Reproduce:**
1. Go to checkout on iPhone 14 Safari
2. Tap the "Card Number" field (keyboard opens)
3. Scroll to bottom to find Place Order button

**Expected Result:** Button visible above keyboard or page scrolls correctly  
**Actual Result:** Button completely hidden; user cannot tap it without dismissing keyboard first and losing field focus
